# Ordinal Encoding:
-------------------




### **IMPORTANT**:

Ordinal encoding only works for categories that have an order. 
E.g: 

1. ['Low', 'Medium', 'High'] has an order: 
    
    Low < Medium < High

2. ['Masters', 'Bachelors', 'High School', 'PhD'] has an order: 

    High School < Bachelors < Masters < PhD
